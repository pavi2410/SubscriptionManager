## Tech Stack

Built with [Ionic React](https://ionicframework.com/react)


## Run Locally

Clone the project

```bash
git clone https://github.com/pavi2410/Subscrypt
```

Go to the project directory

```bash
cd Subscrypt
```

Install dependencies

```bash
npm install
```

### To run on Web

Start the server

```bash
ionic serve
```

Go to http://localhost:8100

### To run on Android/iOS

```bash
ionic run android
```
